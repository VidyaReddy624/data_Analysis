# Data Analysis using Python

## 📌 Overview
This project demonstrates the process of analyzing a dataset using Python. It involves data loading, cleaning, exploration, visualization, and extracting meaningful insights. The project helps understand patterns, trends, and relationships within the data to support data-driven decision-making.

---

## 🚀 Features
- Load dataset using Pandas
- Data Cleaning
- Handle Missing Values
- Remove Duplicate Records
- Statistical Summary
- Data Visualization
- Correlation Analysis
- Extract Key Insights

---

## 🛠️ Technologies Used
- Python 3
- Pandas
- NumPy
- Matplotlib

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Data-Analysis.git
```

Navigate to the project folder:

```bash
cd Data-Analysis
```

Install the required libraries:

```bash
pip install pandas numpy matplotlib
```

---

## ▶️ How to Run

Run the Python script:

```bash
python data_analysis.py
```
